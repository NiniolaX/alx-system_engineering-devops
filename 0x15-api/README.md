##0x15 - API
For this project, we work with APIs, fetching data from them and formatting
the data received.

### Project files
1. 0-gather_data_from_an_API.py - Uses this [REST API](https://jsonplaceholder.typicode.com/) for
	a given employee ID and returns information about his/her TODO list progress.
2. 1-export_to_CSV.py - Extends [Script-0]() to export data in CSV format.
3. 2-export_to_JSON.py - Extends [Script-0]() to export data in JSON format.
4. 3-dictionary_of_list_of_dictionaries.py - Extends []() to export data of
	employees in JSON format.
